# 🚀 Trabalho Final de Front-End — Serratec

Projeto desenvolvido como trabalho final da matéria de Front-End, com foco na aplicação prática dos fundamentos do desenvolvimento web utilizando **HTML, CSS e JavaScript**.

O objetivo principal do projeto foi criar um site educacional e interativo voltado para o aprendizado introdutório de tecnologias front-end, proporcionando uma navegação simples, organizada e intuitiva.

## 📚 Sobre o Projeto

O site funciona como um mini guia introdutório de desenvolvimento front-end, apresentando conteúdos básicos sobre:

* HTML
* CSS
* JavaScript

Além disso, o projeto conta com páginas institucionais como:

* Home
* Sobre
* Contato

Cada integrante da equipe ficou responsável pelo desenvolvimento de uma página específica, permitindo a prática de trabalho em equipe, organização de código e separação de responsabilidades.
