Primeiros conceitos

HTML é a linguagem que vai estruturar o seu codigo para a web, é como o esqueleto que mantém a página. Ela é montada a base de tags, que são os elementos que vão definir o conteudo da página. Veremos algumas das principais a seguir.


Principais Tags


Tags de texto

Para as tags de texto temos o <h1> até o <h6> para definir títulos, temos também a tag <p> para definir parágrafos e textos gerais e a <br> para quebra de linhas e formatação do texto.


Listas e links

Na construção de listas teremos duas opções, sendo elas a tag <ol> e a <ul>, a primeira criando listas ordenadas, sendo indicada por números, já a segunda seriam listas não ordenadas, indicadas por uma pequena bola. Para atribuir o link de outro ou um campo que leve a outro ponto do proprio site usamos a tag <a>, que traz junto dela o atributo "href" que direciona o link.


Estruturais

Essas são as tags que vão organizar e montar nossa página, para começar temos a tag <body> que é onde fica tudo que o usuario vai conseguir ver da nossa página, a <header> é quem montar nosso cabeçalho e a <footer> é quem faz o papel do rodapé. Entre elas temos também a tag <main> que sera o conteudo principal da página, as <div> que são conteineres genericos para dividirmos conteudos dentro do main e a <nav> onde podemos criar um menu de navegação.
